Project Title

Ebundle Master

Vision Statement

An e-ticketing system that allows customers to purchase concert and performance tickets, select date and location of concert, purchase ticket.In addtion the system offers bundles to customers which allows them to search for transport such as bus and train, hotel room and restaurant.

Introduction

As a group we first came up with the idea of an eticket system. After consulting with our lecturer we decided we need to add extra features such as offering to search for Hotel, Transport and Restaurant. We then name our system EBundleMaster. We first came up with a list of brief use cases, then we divided among each member of the team. I was tasked to write use cases for the following:

  • Account Log-In
  • Search for concert/performance genre
  • Search for restaurant
  • Search for restaurant type
  • Book Restaurant
  • Confirm Booking Detail
