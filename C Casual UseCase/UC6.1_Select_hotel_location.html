Casual Use Cases:

UC6.1: Select hotel location

Use Case Number: 6.1
Use Case Name: Select hotel location
Success: When the customer invokes the search hotel option, it will display a location input box for customer to input location they wish to stay, the system will search for hotel.
Alternative Scenario:
The customer enters a invalid location, in which case customer must re-enter valid location. The customer does not want to book for hotel rooms, in which case customer can directly go pay for the ticket or exit the system.
