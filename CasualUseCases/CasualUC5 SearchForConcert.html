Casual Use Cases:

UC ID 5: Search for Concert

Use Case ID: 5
Use Case Name: Search for Concert
Success: Customer successfully able to search for a concert & thereafter is able to search concert performance genre, date, or location
Alternative Scenario:
  • No concert: The system returns with an error message
  • No concerts of specified genre: The system returns with an error message & brings up other concerts genres
  • No concert on date specified: The system returns with an error message & brings up other concerts dates close to the specified date
  • No concert in location specified: The system returns with an error message & brings up other concerts locations
