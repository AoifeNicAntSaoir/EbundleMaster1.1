Casual Use Cases:

UC ID 8.3 - Select arrival location

Use Case Number: 8.3
Use Case Name: Search arrival location
Description: A customer chooses an arrival location for their transport of choice
Alternative Scenario:
  • A customer chooses no arrival location - the system shows up an error message
  • The date available does not arrive in the arrival location - the system shows up an error message & brings up other arrival locations on the date specified
