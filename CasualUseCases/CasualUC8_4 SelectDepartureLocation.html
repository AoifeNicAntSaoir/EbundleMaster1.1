Casual Use Cases:

UC ID 8.4 - Select departure location

Use Case Number: 8.4
Use Case Name: Search departure location
Description: A customer chooses a departure location for their transport of choice
Alternative Scenario:
  • A customer chooses no departure location - the system shows up an error message
  • The date available does not arrive in the departure location - the system shows up an error message & brings up other departure locations on the date specified
